# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

## Navigate

```bash
cd portfolio
```

## Install Packages

```bash
npm install
```

## Run

```bash
ng serve
```

Open

```
http://localhost:4200
```

---

# 📦 Production Build

```bash
ng build --configuration production
```

---

# 🚀 Deployment

This portfolio is deployed using **Netlify**.

To deploy:

```bash
ng build
```

Upload

```
dist/portfolio/browser
```

to Netlify.
